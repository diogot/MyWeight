# MyWeight

MyWeight is a body mass tracker focused on make easy to input new data and check your weight history.


## Getting Started

### Prerequisites

- Ruby
- Bundler

### Installing

Just run on the project root

```
$ rake setup
```


## Running the tests

```
$ rake unit_tests
```
